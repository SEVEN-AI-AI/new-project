# 物流时效与头程费用分摊系统

这是一个纯静态前端系统，基于 `平均头程费.xlsx` 的两个模块整理：

- Sheet1 物流时效录入：支持平台、店铺、服务商、渠道等字段选择或手动输入。
- 时间节点自动计算：送仓、开船、到港、卡派、海外仓到货后自动计算各阶段时效。
- 物流时效追踪：展示每票从送仓到海外仓的全链路时效。
- 头程费用核算：筛选条件会同步影响费用列表和统计卡片。
- 服务商表现分析：比较平均时效、成本、延误票和风险票。
- 延误预警：识别实际超承诺和在途风险票。

## 本地打开

直接用浏览器打开 `index.html`。

## GitHub Pages 部署

项目是静态文件，推送到 GitHub 后可以开启 GitHub Pages，部署来源选择 `main` 分支根目录即可。
